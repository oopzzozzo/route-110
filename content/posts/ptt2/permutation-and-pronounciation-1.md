以下是我整理過的版本：<br>
基本上中文句字可以畫成二元樹，例如：<br>
```
    /\
   /  \
  /\  /\
 /\上 /  \
基本 /    \
   /\     \
  /  \    /\
 /\  /\  /  \
中文句子 /   /\
       /   /  \
      /\  /\  /\
     可以畫成/\ 樹
           二元
```
然後每個節點合併的時候，如果左右子樹的最右左葉都還是三聲，<br>
左子樹的最右葉就要變二聲。<br>
