開始上班一年半，side project 的產量為零。為免淪至 #1VaoB7DJ，總得找些事做做。
於是想說我應該來整合一下我的紅外線遙控冷氣、螢幕和多段電燈。

直覺是用手機控制所有東西，甚至已經有現成 app ，只要選各電器的場牌就能用。
可惜我的手機沒有紅外線發射器，只能買零件 DIY。
我決定用電燈的六鍵小遙控器控制，因為我不想學寫 apk，也不想因此買 switch 搖桿。
於是上網買了基本的 Arduino Nano 和紅外線收發模組。

我玩了玩 arduino，雖然能正確解讀其他遙控器的編碼，卻無法發出訊號。
由於我沒有第二個收發模組可以測試，只能用手機鏡頭照 LED 看它始終是微亮。
為了 debug，我又上網買了一個 4 𦁈（可能比 9v 電池還便宜）的三用電表。
結果買回來發現我沒有小螺絲起子可以裝電表的電池，至今還沒開始 debug。
我到回台前才剛好路過間小五金，用英文買到螺絲起子。（新加坡應該不叫羅賴把…）

去年底銷假時，我想說應該做點新加坡才能做的事。
就跑去搭 50 樓公宅的電梯，測測加速度，看看越高的電梯是否越晃。
收數據時想說應該要收一下「氣壓」數據當作高度參考。
結果我手機太便宜沒有氣壓計。只取得加速度資料就不太好用。

綜上兩個 project 遇到的問題，好像都是換手機就秒解的。
問題 -> 換手機 -> 錢 -> 工作，怎麼最後又回到原點了？
資本主義好可怕。
