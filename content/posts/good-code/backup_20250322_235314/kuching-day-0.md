適逢年底銷假，打算回台灣過年前先找個地方玩。
然後我又很懶得安排，就變成放假後才決定要去哪。
機票便宜的地方大致有曼古馬尼拉和砂勞越。
儘管假有十天，我決定時大概剩七天，另外我還有其他事可做，不一定要玩七天。
泰國和菲律賓要約一千五的簽證費，之後去一次玩久一點也不錯，就決定先去東馬。

28 日晚上七點多降落古晉。海關排隊時連上網路、打開疫苗和財力證明。
官：「台灣人，這要簽證嗎？」旁：「不用。」
官：「你會講英文嗎？」我：「嗯」
官：「你一個人啊？」我：「對，一個人。」
我：「嗯�� 我二十五歲了。」官：「哇～對不起，我真蠢。」

結果拍照按指紋就蓋入境章了，沒看其他文件。

我還沒換馬幣，但機場人員表示機場只有 ATM 沒換錢。
晚上八點店都差不多關了，我於是先叫 Grab 到旅館。
Grab 司機阿姨：「我女兒在台灣說過年要賺 OT（加班費），我年後去台灣找她。」
到旅館後因為手上沒有現金，所以也無法去河邊逛夜市，只能在房間查行程。
決定隔天去砂勞越文化村，後天去峇哥國家公園。
